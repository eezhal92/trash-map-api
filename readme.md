# Trash Map API

## Requirement
* Node >=v6.9.1
* MongoDB v3.4.0
* Yarn

## How to set development
* Install all dependecies `yarn install`
* Copy `.env.example` into `.env` and change the env variable values
* Run the app `yarn run dev`


## Run for Production
* Build the source `yarn run build`
* Run the app `yarn run start`
